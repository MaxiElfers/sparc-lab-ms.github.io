# Site images

Drop the following files in this folder before the first production build:

- `sparc-logo.png` — site logo shown in the header (transparent PNG, ~400×120)
- `favicon.ico` — browser tab icon
- `sparc-hero.jpg` — hero image on the homepage (referenced in `content/_index.md`)
- `sparc-share.png` — social-sharing preview (1200×630)

Team portraits live next to each member's `index.md` under `content/team/<slug>/`. Rename the file `featured.jpg` to have it picked up by the Apéro project list-grid layout.
